# Fullscreen menu
versie 2.0

Onlangs op de website hellfest.fr beland en vond de manier dat ze de menu weergeven erg mooi. Dat heeft me geïnspireerd om zelf aan de gang te gaan met Full Screen Menu’s. Na wat research op internet over fullscreen menu’s ben ik gaan kijken hoe andere site’s full screen menu’s maken. Vond uiteindelijk die van hellfest.fr het mooist, en ben toen gaan proberen deze zo goed mogelijk na te maken.

Ik dacht altijd dat je voor een fullscreen menu heel veel javascript code nodig had, maar dat viel best wel mee te vallen. Het enige javascript wat er voor nodig is is wanneer er op een specifieke knop gedrukt wordt, dat er een class toegevoegd wordt aan de fullscreen menu div. Zo kun je in CSS de styling toevoegen voor wanneer hij uitgeklapt is (dus een specifieke class heeft), en styling toevoegen voor wanneer hij een class mist.

Ik wou niet dat het een hele statige popup werdt, maar juist dat hij mooi geanimeerd was. Zo wil ik namelijk dat hij mooi in-fade’t. Items in het menu op een mooie manier getoond worden. Wat ik ook wil is dat het hamburger menu door middel van animaties vervormd wordt tot een kruisje. Ziet er een stuk mooier uit dan een harde (geen) overgang.

Qua styling er op gelet dat hij makkelijk aan te passen is, zodat ik hem gemakkelijk kan hergebruiken in andere projecten, zonder veel moeite te hoeven doen om hem in de stijl van de website te laten passen. Het Fullscreen menu is gemaakt in GruntJS.

Ben uiteindelijk erg tevreden over het eindresultaat. Op gebied van animaties is het nog niet helemaal wat ik wil, maar goed, daar was deze POC ook niet voor bedoeld. Wil deze menu vorm ook implementeren in mijn portfolio website.

  - Toepassen van simpele animaties
